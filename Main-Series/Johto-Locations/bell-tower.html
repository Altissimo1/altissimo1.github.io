Bell Tower

Known as Tin Tower in Gold, Silver, and Crystal.

Access to:
Exit on 1F: Ecruteak City

Access to:
Exit on 1F: Bellchime Trail

Walking (all floors)

Static

All Versions
Pokémon Level Games Notes
Ho-Oh 40 G S C On 10F. It can only be encountered here after obtaining the Rainbow Wing at the Goldenrod Radio Tower.
If defeated, no other Ho-Oh can be obtained on the save file unless it is restarted.
It always holds a Sacred Ash.
Ho-Oh 70 G S C On 10F. It can only be encountered here after obtaining the Rainbow Wing in Pewter City.
If defeated, no other Ho-Oh can be obtained on the save file unless it is restarted.
It always holds a Sacred Ash.
Ho-Oh 60 G S C On 10F. It can only be encountered here after obtaining the Rainbow Wing here.
If defeated, no other Ho-Oh can be obtained on the save file unless it is restarted.
It always holds a Sacred Ash.
Suicune 40 G S C On 1F. It can only be encountered here after first releasing it from the Burned Tower and obtaining the Clear Bell at the Goldenrod Radio Tower.
If defeated, no other Suicune can be obtained on the save file unless it is restarted.

Pokémon

  • Wild Rattata have a 5% chance of holding a Chilan Berry.
  • Wild Zigzagoon have a 5% chance of holding an Oran Berry.
  • Wild Spinda have a 5% chance of holding a Chesto Berry.
  • Wild Chatot have a 5% chance of holding a Metronome.
