Safari Zone

Access to:
South: Safari Zone Gate

Utility:
Surf can be used to encounter wild Pokémon.

Overview

The Johto Safari Zone is unique among Safari Zones in that the player has some control over the Pokémon species that can appear. There are two methods by which the player can alter the Safari Zone, which are unlocked by completing challenges for Baoba, the Safari Zone Warden.

Areas

There are six areas in the Safari Zone, arranged in two rows of three areas. The player enters the Safari Zone in the bottom middle area. Each area can have one of twelve area types, shown in the table below.

Plains Meadow Savannah Peak
Rocky Beach Wetland Forest Swamp
Marshland Wasteland Mountain Desert

When the player first visits the Safari Zone, the areas will be arranged in one of ten possible default layouts, shown below. All ten default layouts have the Peak area type as the bottom middle area. After Baoba issues his second challenge, the player will get the option to move and replace the default area types with any others from the list above. Each area type can only be used for one area in the Safari Zone. Some area types feature water, where the player can encounter Pokémon with Surf and Fishing Rods.

Blocks

Blocks can be added to an area to affect their encounter tables. The blocks, and their categories, are given below.

Category Block
Plains Shrubbery Red Flower White Flower
Forest Tree Stump Branches
Rocks Small Rock Big Rock Mossy Rock
Water Puddle Fountain Water Hole
Other Guidepost (R) Guidepost (L) Trash Can
Long Fence 1 Long Fence 2 Your Statue
Bench Your Flag Outside Lamp
Short Fence 1 Short Fence 2 Signboard

Up to 30 Blocks can be placed in an (active) area at a time. If Blocks in categories other than 'Other' are placed in an area type, they can affect the encounter tables for that area type. Each area type has a default set of Pokémon that can spawn, and a secondary set of Pokémon, where each encounter slot is tied to a given number of Blocks. Placing Blocks of the given type in the area type will enable the Pokémon from the secondary set to begin appearing once their number of Blocks has been reached.

Each encounter slot in the secondary set is actually tied to a given number of Block points, which is not the same as the physical number of Blocks placed in the area type. It is possible to upgrade area types so that physical Block objects count for more than one Block point, up to a maximum of seven points per Block. Upgrading is required to get some Pokémon to spawn, as some species require more than 30 Block points, and only 30 Blocks may be placed in an area.

Each area type is upgraded separately. An area type is upgraded by having it as the type for an active area for a given number of real-time days (incrementing every midnight). If an area type is swapped out, it will remember the number of days it has been active and begin counting from that number the next time it is placed as an active area. Therefore, areas can never be downgraded.

As area types are upgraded, Blocks in non-'Other' categories placed in the area type will offer more than one Block point per Block. No Pokémon relies on 'Other' blocks to spawn, making them useless for altering encounter tables. Each of the four usable Block categories requires the area type to have been active for a different number of days. For instance, upgrading Plains Blocks to their second level (where one Block equals two Block points) requires the area type to be active for 10 days, but upgrading Water Blocks requires 40 days. The number of days required to reach each upgrade level is given below. The numbers in the first row refer to the number of Block points per Block that will be obtained at each upgrade level.

Category 2 3 4 5 6 7
Plains 10 50 100 150 200 250
Forest 20 60 110 160 210 250
Rocks 30 70 120 170 220 250
Water 40 80 130 180 230 250

Unlocking All Features

Being able to add Blocks and change area types are not unlocked by default when the player enters the Safari Zone. Instead, the player will have to complete challenges for Baoba, make story progress, and play the game for several hours to unlock features.

  1. Upon first arriving at the Safari Zone, Baoba will issue the challenge of catching a Geodude in the Safari Zone. Geodude can be found in the Peak area type, and all ten of the default layouts have Peak as the entry point.
  2. After the player comples Baoba's first challenge, the player must accumulate three hours of in-game play time, at which point Baoba will call to offer his second challenge. In this challenge, the player will be allowed to swap and rearrange area types of the areas. Using this ability, the player must add the Desert area type and catch a Sandshrew there.
  3. After the player has defeated the Elite Four, and accumulated three hours of in-game play time since completing Baoba's second challenge, Baoba will call the player to give them six Blocks, and the player can begin using them. For every three hours of in-game play time accumulated afterward, Baoba will call the player to provide six additional Blocks, until he has provided all 24.

Pokémon

The following tables give information about the default and secondary sets of encounters. Encounters are ordered into sets of ten (except for Surfing/Fishing secondary sets, which have a smaller amount of encounters), with the "Slot" cell showing the order that Pokémon are internally stored in the set. Every encounter has an equal 10% chance of appearing.

It is possible to have some of the requirements for Pokémon in the secondary set met, but not all at once, meaning the actual encounter table can be a mix of Pokémon from the default and secondary sets. The "Slot" property of each encounter slot is used to determine which default encounters will be overwritten with which secondary encounters.

For example, imagine that the player is in the Meadow area, and has fifteen Plains Block points and five Rocks Block points. This means the player has met the requirements to spawn Raticate (5 Plains Block points), Chansey (12 Plains Block points), and Geodude (3 Rocks Block points). These three encounters will replace the first three Slots in the default set: lv.15 Jigglypuff, lv.15 Hoppip/Sunkern/Marill, and lv.15 Hoppip/Hoppip/Wooper. The order in which they replace these slots is determined by the secondary set's Slot property, with the lower Slot number of the secondary set replacing the earlier encounter on the default set. Therefore, Raticate (slot 0) will replace Jigglypuff, Chansey (slot 1) will replace Hoppip/Sunkern/Marill, and Geodude (slot 3) will replace Hoppip/Hoppip/Wooper, in that order.

Surfing encounters have three Pokémon in the secondary set, so the last seven encounters of the default Surfing set will never be overwritten. Fishing encounters have two Pokémon in the secondary set, so the last eight encounters of the default Fishing set will never be overwritten.

Resources

As the last upgrade for an area unlocks after 250 days of real time - over 8 months! - some players have put together suggestions and guides for efficiency in dealing with the Safari Zone.

  • Bulbapedia's page, in addition to serving as the primary source for most of the prose on this page, also has a table that displays the days to unlock every species exclusive to the Safari Zone, as well as the days needed to unlock each evolutionary family (assuming the player will evolve or breed to get the missing members of the family). For each area type, it also lists some more specifics about the method to unlock unique species. Although the information about how to maximize upgrades is likely accurate, as of this writing, the actual encounter tables have some differences from the source I used for the tables (PokemonSlots), which I believe to reflect inaccuracies on Bulbapedia's part.
  • This guide, written by the user Giant Octopodes on Bulbagarden, also has some information about minimizing the days required for each area.
  • This guide on GameFAQs, written by the user marter, combines the information to show how many blocks and how many days are required to unlock each species, reducing the amount of calculation the player might have to do as to how many blocks are needed.

In some of these sources, alternate names may be used to refer to some of the Block categories. Bulbapedia uses "Peak" instead of "Rocks" and "Waterside" instead of "Water", while the GameFAQs guide uses "Grassland" instead of "Plains" and "Rocky" instead of "Rocks".
