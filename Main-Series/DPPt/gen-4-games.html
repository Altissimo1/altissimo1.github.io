Generation 4 Games

The following table contains every Pokémon and the games in Generation 4 (Diamond, Pearl, Platinum, HeartGold, SoulSilver, and the Pokéwalker) that it can be obtained in.

For specific encounter locations in each game, see this page for Diamond, Pearl, and Platinum, this page for HeartGold and SoulSilver, and this page for the Pokéwalker.

All 493 Pokémon can be obtained between these games. However, Regirock, Regice, and Registeel are only obtainable with access to a Regigigas distributed in a now-defunct event, and Regigigas itself can only be obtained outside events with Regirock, Regice, and Registeel in the party. Without access to the event Regigigas, the three Hoenn Regis need to be transferred from Ruby, Sapphire, or Emerald. See the Ruby, Sapphire, and Emerald locations page for details about where to find them.

An asterisk * next to the letter for Diamond, Pearl, or Platinum indicates that it requires using the dual-slot DS functionality with a GBA Pokémon game. See the Diamond, Pearl, and Platinum Locations page for further details.

An asterisk * next to the Pokéwalker letters indicates a Pokémon that can be evolved or bred from a Pokémon found in the Pokéwalker.
