Ramanas Park

Access to:
Exit: Route 221

Utility:
Surf is required to access some of the rooms (and therefore the Legendary Pokémon inside).

Pokémon

Static

Discovery Room

A Discovery Slate is required to initiate a battle with a random Legendary Giant.

All Versions
Pokémon Level Notes
Regirock 70 If defeated or run from, it can be encountered again by using another Discovery Slate.
The player will receive a Regirock Statue after capturing it.
Regice 70 If defeated or run from, it can be encountered again by using another Discovery Slate.
The player will receive a Regice Statue after capturing it.
Registeel 70 If defeated or run from, it can be encountered again by using another Discovery Slate.
The player will receive a Registeel Statue after capturing it.

Distortion Room

A Distortion Slate is required to initiate a battle with Giratina. The Distortion Slate can only be purchased after catching Regirock, Regice, Registeel, and Giratina (at Turnback Cave).

All Versions
Pokémon Level Notes
Giratina (Origin) 70 It cannot be caught.
The player will receive a Giratina Statue and a Griseous Orb after defeating it.

Soul Room

A Soul Slate is required to initiate a battle with a random one of Latios and Latias. The Soul Slate can only be purchased after catching Regirock, Regice, and Registeel.

The player will receive the Soul Dew for catching both Latios and Latias.

All Versions
Pokémon Level Notes
Latios 70 If defeated or run from, he can be encountered again by using another Soul Slate.
The player will receive a Latios Statue and an Apicot Berry after capturing him.
Latias 70 If defeated or run from, she can be encountered again by using another Soul Slate.
The player will receive a Latias Statue and a Petaya Berry after capturing her.

Johto Room

The Johto Room is only accessible in Brilliant Diamond. A Johto Slate is required to initiate a battle with a random Legendary Beast. The Johto Slate can only be purchased after catching Regirock, Regice, and Registeel.

Brilliant Diamond
Pokémon Level Notes
Raikou 70 If defeated or run from, it can be encountered again by using another Johto Slate.
The player will receive a Raikou Statue after capturing it.
Entei 70 If defeated or run from, it can be encountered again by using another Johto Slate.
The player will receive an Entei Statue after capturing it.
Suicune 70 If defeated or run from, it can be encountered again by using another Johto Slate.
The player will receive a Suicune Statue after capturing it.

Kanto Room

The Kanto Room is only accessible in Shining Pearl. A Kanto Slate is required to initiate a battle with a random Legendary Bird. The Kanto Slate can only be purchased after catching Regirock, Regice, and Registeel.

The player will receive a Moltres Statue after capturing it.

Rainbow Room
