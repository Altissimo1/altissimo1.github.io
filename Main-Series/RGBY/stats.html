Red, Green, Blue, and Yellow Stat Generation

In Red, Green, Blue, and Yellow, when a Pokémon is generated, it will have DVs generated between 0 and 15 for its Attack, Defense, Special, and Speed stats. The HP DV is determined using the DVs of the other four stats, once they have been generated. A proper random generation of stats would allow for any configuration of the four main stat DVs to be obtainable on any Pokémon, but the method by which the game generates stats means that, in reality, not all spreads are available on every Pokémon. This page explains the process and shows what possibilities are available for what Pokémon.

Overview

When the game is running, it normally tracks two numbers in memory, which we we call 'RandomAdd' and 'RandomSub'. On every frame, the game will update them using a hardware timer, which we call 'rDiv'. This timer will increment regardless of what the player is doing in the game, meaning the player has no control over it at any point.

When the player is in an area in which they can get wild encounters, the game will generate a wild encounter as follows, using three consecutive values from RandomAdd.

  1. The game checks if an encounter should happen at all.
  2. The game determines the Speed and Special DVs for the encounter.
  3. The game determines the Attack and Defense DVs.

Because RandomAdd is always updated using the ever-increasing rDiv number, and because all three numbers are taken from RandomAdd at one time, this effectively means that the results of the DVs generated in steps 2 and 3 are linked to the result of step 1 - whether an encounter happens at all. In order for an encounter to occur, the starting RandomAdd value must be less than the encounter rate for the area. Since, therefore, only a few small possible values for RandomAdd will ever trigger an encounter, only a few small possible values will be used for generating the DVs, rather than the entire range of possible RandomAdd values.

This incidentally has the effect that areas with higher encounter rates provide more possible DV combinations, since the value of RandomAdd has a greater range.

The short version is that the possible range of DVs a Pokémon can have is tied to the encounter rate of the area(s) it is found in.

Encounter Rates

The above explanation indicates that areas with the same encounter rate will have the same possible DV spreads available for Pokémon that can be found there. The tables below indicate which encounter rate is used by every location that can have wild encounters, and what Pokémon are available there. The lowest possible level for each Pokémon (and its evolution) is also provided, which can help in identifying, say, whether the Pokémon is legal for one of the Stadium cups or with a certain move.

Fishing encounters do not use this method of generation. They are part of the "Other encounter types" discussed below.
