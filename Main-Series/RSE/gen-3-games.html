Generation 3 Games

The following table contains every Pokémon and the games in Generation 3 (Ruby, Sapphire, Emerald, FireRed, LeafGreen, Colosseum, XD) that it can be obtained in.

For specific encounter locations in each game, see this page for Ruby, Sapphire, and Emerald, this page for FireRed and LeafGreen, this page for Colosseum, and this page for XD.

Pokémon in Colosseum or XD with an asterisk * next to the game name are Pokémon that can be bred from a Pokémon obtainable in that game. Since the breeding mechanic is not present in either game, the Pokémon would need to be traded to a Game Boy Advance game to be bred.
