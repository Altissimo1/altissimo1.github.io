Ruby, Sapphire, and Emerald Wild Held Item Locations

The following table contains a list of every Pokémon that can hold an item when encountered in the wild.

Each item is assigned to either a 5%, 50%, or 100% slot on the Pokémon, which indicates how likely it is that any given Pokémon of that species will have the item. A Pokémon can have items assigned to both 5% and 50% slots.

This table only shows Pokémon that can actually be encountered in the wild in Ruby, Sapphire, and Emerald. For example, Raichu has hold item data assigned to it, but no Raichu can be encountered in the wild, so Raichu is not in this list.

Below the main table is also a list of the items held by wild Pokémon appearing in Altering Cave, which has special data for its inhabitants' held items.

The wild Latias or Latias encountered on the event-only Southern Island always holds the Soul Dew.
