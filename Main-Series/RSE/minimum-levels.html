Ruby, Sapphire, and Emerald Minimum Levels

This page lists the lowest possible level at which Pokémon can be obtained in Ruby, Sapphire, and Emerald alone.

Any Pokémon that can be obtained in an Egg can be obtained at level 5. This first table will show only the levels that can be obtained without the use of breeding, except for Pokémon that can only be obtained via breeding. A list of all breedable Pokémon, and evolutions that can be obtained at a lower level by using breeding, is given below the primary table.
