FireRed and LeafGreen Rereleases

On 20 Feb, Nintendo revealed that FireRed and LeafGreen will be re-released on Nintendo Switch, with support for local multiplayer only. Although early documentation about this release suggested the games will be Home-compatible, that information was removed from the documentation quickly. It is yet unclear why this has happened, and it remains to be seen what, if any, compatibility these rereleases will have with other Pokémon titles. As more information gets revealed, I'll update my existing FireRed and LeafGreen documentation accordingly.

If there is a FireRed/LeafGreen resource you want to see referenced or want to see me make, or if you find any problems with my existing information, please let me know or consider submitting a PR or starting an Issue on GitHub.

Games

Main Series: "Core" series games. Currently contains pages for the following games:

  • Red, Green, Blue, and Yellow
  • Gold, Silver, and Crystal
  • Ruby, Sapphire, and Emerald
  • FireRed and LeafGreen
  • Diamond, Pearl, and Platinum
  • HeartGold and SoulSilver
  • Omega Ruby and Alpha Sapphire
  • Let's Go Pikachu and Let's Go Eevee
  • Brilliant Diamond and Shining Pearl

Supplementary Series: A term I made up to describe games that don't have the exact same gameplay loop as core games, but can trade and interact with them. Currently contains pages for the following games:

  • Legends series
  • Stadium series
  • Orre games (Colosseum and XD)

Side Series: Other Pokémon games with altogether different mechanics. Currently contains pages for the following games:

  • Pinball series
  • Mystery Dungeon series
  • Ranger series
  • Trozei / Link series
