Mystery Gift

The player can purchase items and Pokémon using the Poké Coupons collected as a reward for clearing colosseum challenges, which can be sent to a connected copy of the DS games Diamond, Pearl, Platinum, HeartGold, or SoulSilver using Mystery Gift. This page shows which items and Pokémon are available for purchase this way.

Pokémon

The following Pokémon can be selected in the Mystery Gift menu and sent to a connected DS game. All of them cost 0 Poké Coupons, but require certain conditions to be met before they can be unlocked.

Surfing Pikachu

Surfing Pikachu is available as "Secret Gift" in the Mystery Gift menu, and becomes available the first time the player completes Stargazer Colosseum. The gift can be sent many times, but a single save file on a connected DS game can only receive it one time.

Electivire

Electivire is available as "Shocking Secret Gift" in the Mystery Gift menu. This becomes available if the player enters a specific code in the "Self-Introduction" option in the Profile menu. The code needs to be saved in order to activate the gift, and the code is case-sensitive. This gift is not available in the Japanese release.

Unlocking Electivire also gives the player the Gold Pass.

Magmortar

Magmortar is available as "Heated Secret Gift" in the Mystery Gift menu. This becomes available if the player enters a specific code in the "Self-Introduction" option in the Profile menu. The code needs to be saved in order to activate the gift, and the code is case-sensitive. This gift is not available in the Japanese release.

Unlocking Magmortar also gives the player the Silver Pass.
