Sunny Park Colosseum Pokémon (Doubles)

These Pokémon can be selected for opposing Trainers' teams in Sunny Park Colosseum for Double Little Battles. Little Battles are only available in this colosseum after completing Stargazer Colosseum for the first time.

All Pokémon have random IVs and Natures. All Pokémon are level 5.

Moves marked as "random level-up moves" should theoretically be only taken from the moves the Pokémon can learn before reaching level 5, but I'm unsure if this is exactly how this works in practice; for instance, Ralts' moveset already has Growl, which is its only level-up move until level 6. If you have any further information, please contact me. Information about TMs included in the different random TM groups can be found here.

At any given rank, the player may see Pokémon whose rank is the same as the current rank number, one lower, or one higher. For example, at Rank 3, Pokémon in ranks 2-4 can be encountered.
