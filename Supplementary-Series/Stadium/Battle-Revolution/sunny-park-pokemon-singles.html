Sunny Park Pokémon (Singles)

These Pokémon can be selected for opposing teams in the Sunny Park Colosseum for Little Battles. Little Battles are only available in this colosseum after completing Stargazer Colosseum for the first time.

All Pokémon are level 5 and have random IVs and Natures. Moves marked as "random level-up moves" should theoretically be only taken from the moves the Pokémon can learn before reaching level 5, but I'm unsure if this is exactly how this works in practice; for instance, Ralts' moveset already has Growl, which is its only level-up move until level 6. If you have any further information, please contact me.

Information about TM generation can be found here.
