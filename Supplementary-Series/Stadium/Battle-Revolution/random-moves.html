Random Moves

On the pages that contain lists of Pokémon, you may see some Pokémon with moves identified as, for instance, "Random Offensive TM Move 1". These refer to pools of TMs, specially marked in the game data, that will be used to select a move from. The TM move will always be one that the Pokémon can learn (using Diamond & Pearl TMs and TM compatibility) that is taken from the specified group. This page shows which TMs are available in which groups.

When a random TM is generated for a Pokémon, the game will actually select from all possible TMs contained within that group and groups with lower numbers. Therefore, a move whose TM group is listed as 2 can actually select from TM groups 1 and 2, and a TM group listed as 3 or 4 can select from all three TM groups in that category.

Return, Frustration, and Hidden Power are not in any pool, so they will never be used for random TM generation.
