Quest: Cherrim, Cherrim, Come to Us♪

Nectar Victoria on Union Road requests to see a Cherrim. The only place where Cherrim can be found is on the Peril Cliffs.

Obstacles

  • If not already cleared, a vine in the northeastern area requires Cut x1 to clear, which can be found on nearby Gligar, Pinsir, or Staravia. It leads to a wild Cherrim.

Quest: Please Save Shieldon

Shieldon can only be captured as part of this Quest.

Story Pokémon

 Shieldon

Mission 10: Get the Blue Gem!

Obstacles

  • If not already cleared, a vine in the northeastern area requires Cut x1 to clear, which can be found on nearby Gligar, Pinsir, or Staravia. It leads to a wild Cherrim.
  • A Boulder requires Crush x4 to clear, which can be found on a Probopass in Chroma Ruins, which in turn requires Psy Power x3 from a Mismagius in Almia Castle. It helps make the area easier to traverse.
