Obstacles

  • One blue Gigaremo requires Electrify x2 to clear, which can be found on Luxio in the room to the south.
  • One Yellow Switch requires Tackle x2 to clear, which can be found on Machoke or Purugly in a nearby room.
  • One yellow Gigaremo requires Crush x2 to clear, which can be found on Makuhita or Drowzee in a nearby room.
  • One Green Switch requires Tackle x2 to clear, which can be found on Machoke or Purugly in a nearby room.
  • One Electric Door requires Electrify x1 to clear, which can be found on Voltorb.
  • One Wire Fence requires Cut x2 to clear, which can be found on Raticate in a nearby room. It leads to a wild Electabuzz and Magmar.
  • The dark room where Barlow is held captive requires Flash to clear, which can be found on Electabuzz in a nearby room.
  • The Rope binding Barlow requires Cut x1 to clear, which can be found on Raticate in a nearby room.
  • The Kingston Valve requires Tackle x2 to clear, which can be found on Machoke in a nearby room.
