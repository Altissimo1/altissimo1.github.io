Pokémon Ranger

Pokémon Ranger was released in 2006 (outside of Europe) and 2007 (Europe) for the Nintendo DS.

This game contains 213 of the first 386 Pokémon, and an event mission also provides one-time access to a Manaphy egg, which can be transferred (one-way) to a copy of the Nintendo DS games Diamond, Pearl, Platinum, HeartGold, or SoulSilver, using the wireless functionality of the Nintendo DS.

In 2016, Pokémon Ranger was rereleased for the Wii U Virtual Console. Unlike the original releases, this version is not capable of communicating with any other Pokémon game. This means that the Manaphy egg, even if received, cannot be transferred elsewhere.
